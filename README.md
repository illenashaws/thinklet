# Thinklet 🌱
Thinklet is a personal learning project.

I'm a dual degree Information Systems student exploring software development and AI. This project is my way of learning how AI can support decision-making, not only through logic, but also by considering emotions, preferences, values, and human context. At its core, Thinklet is designed to feel alive.

It starts small: remembering a name, greeting you when you return, understanding simple situations, and offering thoughtful suggestions like a quiet companion that listens.
Thinklet is intentionally built step by step. Right now, the focus is understanding fundamentals before adding complexity, building reasoning before intelligence.

## Current Features
- Remembers a user's name
- Tracks last interaction time
- Greets returning users differently
- Supports simple decision-making
- Considers stress and importance levels
- Gives explainable suggestions with reasoning
- Runs entirely in the terminal

## What I'm Learning Through This Project
- Python for AI development
- JSON data handling
- Git and GitHub workflow
- Designing explainable AI behavior
- Human-aware decision systems
- Working toward LLM API integration
- LangChain fundamentals

## Tech Stack

## Vision
Thinklet is being built toward a bigger goal:
An AI companion that supports decision-making while balancing logic, emotions, preferences, and values, while clearly explaining why a recommendation is made.
