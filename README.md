# Thinklet 🌱

Thinklet is a personal learning project.

I’m a dual degree Information Systems student exploring software development and AI.  
This project is my way of learning how an AI can support decision-making — not only through logic, but also by considering emotions, preferences, and values.

At its core, Thinklet is designed to feel alive.  
It starts small: remembering a name, greeting you when you return, responding gently — like a quiet companion that listens.

Thinklet is intentionally built step by step.  
Right now, the focus is on understanding fundamentals before adding complexity.

## Current Features
- Remembers a user's name using a local memory file
- Greets returning users differently from new ones
- Runs in the terminal

## What I’m Learning Through This Project

- Working with LLM APIs
- LangChain fundamentals
- Designing explainable and human-aware AI behavior
- Git and GitHub workflow
